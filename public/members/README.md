# メンバー画像の配置方法

このディレクトリ (`public/members/`) にメンバーのアイコン画像を配置してください。

## ファイル名

`src/data/members/members.json` の `avatar` フィールドで指定されたファイル名と一致させてください：

- `kawasaki.png` - 川崎准也さんのアイコン
- `member2.png` - メンバー2のアイコン
- `member3.png` - メンバー3のアイコン
- `member4.png` - メンバー4のアイコン

## 推奨仕様

- **形式**: PNG または JPG
- **サイズ**: 正方形（例: 400x400px）
- **容量**: 200KB以下推奨

## 画像がない場合

画像がない場合は、プレースホルダーとして絵文字アイコンが表示されます。
