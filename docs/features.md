# 機能説明

## 🏠 ダッシュボード

アプリケーションのメインハブとして、すべての学習モードへのアクセスを提供します。

### 特徴
- **カラフルなタイル**: 各学習モードを視覚的に区別
- **アイコン表示**: 直感的なナビゲーション
- **レスポンシブレイアウト**: デバイスサイズに自動調整

### 利用可能なモード
1. **基本問題** (🟢 緑) - 初級レベルの問題
2. **中級問題** (🟡 黄) - 標準レベルの問題
3. **上級問題** (🔴 赤) - 高度な問題
4. **ランダム出題** (🔵 青) - 全レベルからランダム
5. **RAGボット** (🤖 紫) - AI会話型問題選択

---

## 🤖 RAGボット（AI会話型問題選択）

最も革新的な機能で、自然な会話を通じて最適な問題を選択できます。

### 主な機能

#### 1. 自然な対話
```
ユーザー: こんにちは
ボット: こんにちは！😊
      どのようなトピックの問題に挑戦したいですか？

ユーザー: ネットワークについて学びたい
ボット: ネットワークに関する問題を探しますね！
      
      見つかった問題：
      1. TCPとUDPの違いについて...
      2. OSI参照モデルの7層...
      ...
```

#### 2. 柔軟な問題指定

**トピックで指定**
```
「ネットワーク」
「データベース」
「セキュリティ」
```

**難易度で指定**
```
「簡単なネットワーク」
「上級のデータベース」
```

**問題数で指定**
```
「ネットワーク10問」
「セキュリティ3問」
```

**組み合わせ**
```
「簡単なネットワーク5問」
「上級のセキュリティとアルゴリズム」
```

#### 3. 会話履歴の保持
- 前回の会話を覚えている
- 「もっと問題が欲しい」で追加検索
- 文脈を理解した応答

#### 4. ヘルプ機能
```
ユーザー: ヘルプ
ボット: 使い方をご説明します！📚
      
      1. 学びたいトピックを入力してください
         例：「ネットワーク」「データベース」
      ...
```

### UI/UX
- **ChatGPT風デザイン**: 馴染みやすいインターフェース
- **アバター表示**: ユーザーと AI を視覚的に区別
- **ローディングアニメーション**: 応答待ち時の視覚的フィードバック
- **スクロール自動調整**: 最新メッセージに自動スクロール

## 📝 クイズ画面

### 表示内容
1. **問題文**: Markdown形式でフォーマット
2. **選択肢**: A, B, C, D の4択
3. **コードブロック**: シンタックスハイライト付き
4. **進捗インジケーター**: 現在の問題番号/全問題数

### 解答プロセス
1. 選択肢をクリック
2. 即座に正誤判定
3. 解説を表示
4. 次の問題へ進む

### 解説機能
- **正解の選択肢を表示**
- **詳細な解説文**
- **コードサンプル**（該当する場合）
- **スコア情報**

---

## 📊 結果画面

クイズ終了後に表示される詳細な結果分析。

### 表示内容
1. **総合スコア**: 正解数/全問題数
2. **正答率**: パーセンテージ表示
3. **評価メッセージ**: スコアに応じた励ましのメッセージ
4. **各問題の振り返り**:
   - 問題文
   - ユーザーの回答（正誤表示付き）
   - 正解
   - 解説

### アクション
- **再挑戦**: 同じ問題セットで再度チャレンジ
- **ダッシュボードに戻る**: 別のモードを選択

## 🌐 レスポンシブデザイン

すべての画面がモバイルからデスクトップまで対応。

### ブレークポイント
- **モバイル**: < 640px
- **タブレット**: 640px - 1024px
- **デスクトップ**: > 1024px

### 最適化
- **タッチフレンドリー**: ボタンサイズの調整
- **可読性**: フォントサイズの自動調整
- **レイアウト**: 縦横比に応じた配置変更
