## 🚀 クイックスタート

1. **セットアップ**
   ```bash
   npm install
   cp .env.example .env  # 環境変数を設定
   ```

2. **開発サーバー起動**
   ```bash
   npm run dev
   ```

3. **ブラウザで開く**
   - http://localhost:5173/
