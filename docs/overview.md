# プロジェクト概要

## 🎯 プロジェクト名

**rag-modoki** - RAGプロトタイプ＆ITクイズアプリ

## 📝 概要

rag-modokiは、茨城高専メディア・デザイン・ラボが「得意技・先進技術EXPO＆AI・ICTセミナー」（2025年11月）のサイバーセキュリティ広報啓発ブースで展示するために開発したWebアプリケーションです。

ベクトルデータベースを使わない簡易構成でRAG（検索拡張生成）の流れを再現し、来場者が「検索して根拠を渡してから生成する」というRAGの仕組みをその場で体験できることを目的としています。

## ✨ 主な特徴

### 1. RAG検索ボット

- **RAG検索モード**: アプリ内のMarkdown記事を検索し、その内容を根拠にGeminiが回答を生成
- **クイズレコメンドモード**: 自然な対話からトピック・難易度・問題数を抽出して問題を提案
- **フォールバック機能**: APIが利用できない場合もキーワード検索による基本機能は動作

### 2. 多様な出題モード

- **カテゴリ別出題**: 基本、中級、上級の難易度別
- **ランダム出題**: 幅広い知識をテスト
- **カスタマイズ出題**: RAGボットが提案する問題セット

### 3. 展示テーマに沿った解説記事

- 2025年度のサイバー攻撃事例の比較分析
- オンプレミスとクラウドの比較
- RAGの仕組みの解説
- OpenCVを用いた紙アンケートのスキャン

## 🛠 技術スタック

- **フロントエンド**: React 19 + TypeScript
- **ビルドツール**: Vite
- **スタイリング**: Tailwind CSS + Material UI
- **AI**: Google Gemini 2.0 Flash API
- **Markdown**: react-markdown + react-syntax-highlighter

## 📊 対象ユーザー

- 展示ブースの来場者（中小企業の経営者・一般来場者）
- RAGの仕組みを手を動かして理解したい学習者
